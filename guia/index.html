Instalação

Instale em 3 passos

A skill é uma pasta que você copia para as skills do seu Claude Code.

1

Baixe o repositório

Clone o repositório da skill (ou baixe o ZIP pelo GitHub).

git clone https://github.com/joaopaulobes/wp-plugin-skill.git
2

Copie a skill para o Claude Code

Mova a pasta skill/wp-plugin para o diretório de skills do seu Claude Code.

mkdir -p ~/.claude/skills
cp -r wp-plugin-skill/skill/wp-plugin ~/.claude/skills/
3

Reabra o Claude Code e chame a skill

Pronto! Em qualquer pasta, digite o comando e descreva o seu plugin.

Como usar

Descreva. A skill constrói.

Quanto mais detalhe na descrição, melhor o plugin. Pode comparar com plugins que você já conhece.

1

Chame /wp-plugin

A skill se apresenta e pede a descrição.

2

Descreva o plugin

O que ele faz, onde aparece (painel/site), o que o usuário configura, integrações… e exemplos.

3

Receba e refine

Ela planeja, gera o plugin + zip + auto-update + landing, e você pede ajustes até ficar perfeito.

4

Instale no WordPress

Envie o .zip no painel e ative. Se configurou o auto-update, as próximas versões chegam sozinhas.

Exemplos de descrição

“Um aviso de cookies (LGPD) com texto e cores editáveis no painel e botão de aceitar que lembra a escolha.”
“Avaliações de produtos: um tipo de conteúdo para reviews, nota de 1 a 5, shortcode pra exibir e média no painel.”
“Encurtador de links com redirect, categorias e rastreio de cliques — parecido com o BetterLinks.”

Dúvidas

Perguntas frequentes

Preciso saber programar?

Não. Você descreve o plugin em português e a skill escreve todo o código, no padrão e seguro.

Onde fica a pasta de skills do Claude Code?

Em ~/.claude/skills/. Cada skill é uma subpasta com um SKILL.md. Basta copiar a pasta wp-plugin para lá e reabrir o Claude Code.

Funciona para qualquer tipo de plugin?

Para a grande maioria: telas no painel, front-end, shortcodes, tipos de conteúdo (CPT), REST API, tarefas agendadas, blocos dinâmicos e auto-update. Casos muito específicos podem precisar de ajuste manual — a skill avisa.

A skill copia plugins que já existem?

Não. Você pode comparar com plugins conhecidos para explicar o que quer, mas ela implementa tudo do zero — código de terceiros é protegido; funcionalidade e UX não são.

O auto-update é obrigatório?

Não, é opcional. Se você quiser distribuir o plugin, a skill monta o cliente de update e te entrega o guia completo de como configurar o servidor (basta servir 2 arquivos estáticos).

Os plugins que eu criar são meus?

Sim. A skill é MIT e o que você gera com ela é seu, para usar, vender ou distribuir.

Pronto para criar seu primeiro plugin?

Instale a skill e descreva sua ideia. O resto é com ela.
